Brunch selfmade skeleton
------------------------

Ingrediens:

* ES6
* SASS
* uglifier
* digest
* gzip
* auto-reload

How it was done
---------------

```
> npm init
> npm install --save-dev brunch babel-brunch sass-brunch auto-reload-brunch digest-brunch uglify-js-brunch
```

create ```brunch-config.js```

```brunch build``` or ```npm start``` or ```brunch watch --server```

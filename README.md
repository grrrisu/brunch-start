Brunch selfmade skeleton
------------------------

Ingrediens:

* ES6
* SASS
* uglifier
* digest
* gzip
* auto-reload
* Phaser

Usage
-----

install

```
> npm install
> npm start
> brunch build -p
```

develop

```
> brunch watch -s
```

deploy

How it was done
---------------

```
> npm init
> npm install --save-dev brunch babel-brunch sass-brunch auto-reload-brunch digest-brunch uglify-js-brunch
> npm install --save phaser
```

created ```brunch-config.js```
